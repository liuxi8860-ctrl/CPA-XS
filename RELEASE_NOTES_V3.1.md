# CPA-X V3.1 更新说明

## 本版本重点

- 面板版本统一升级为 `V3.1`
- 移动端布局针对手机做了系统性适配
- 顶部状态栏/底栏沉浸式背景已处理
- 新增第三套主题：`羊毛纸`
- 主题切换改为选择面板模式
- 请求统计改为直接对齐 CPA 当前真实统计
- Token 显示样式优化为更贴近 CPA 的紧凑格式
- 多处危险操作增加确认弹窗说明

## 适合发布的理由

- 已整理为干净可分发副本
- 不包含 `.env`、运行时缓存、数据文件、备份文件
- 保留一键安装脚本、doctor 自动探测脚本、Docker 部署文件
- 已补充适合对外使用的部署说明文档

## 发布前建议

发布到 GitHub 前，建议再次确认：

1. `.env` 没有被包含进去
2. `data/` 没有被包含进去
3. 备份文件没有被包含进去
4. README 中的仓库地址、端口、默认配置与你要公开的版本一致
